# 55 — Evaluation Strategy

**Gateway** · **Wiki Home** · Evaluation Strategy

The operational four-tier evaluation strategy:

1. **Contract tier** — schema/endpoint/identity/span assertions.
2. **Phase tier (M1–M7)** — per-phase evaluators.
3. **Cross-phase / bridge tier (M8)** — MEANTIME bridge + consistency validator.
4. **Regression tier (M9–M10)** — baseline snapshots and CI quality gating.
